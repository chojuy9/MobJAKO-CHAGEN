페이지 공사중

로어북 공유 기능은 현재 개발 중입니다. 더 나은 서비스를 제공하기 위해 열심히 작업하고 있으니 조금만 기다려주세요!

준비 중인 기능

  • 사용자가 직접 만든 로어북 익명 업로드
  • 카테고리별 로어북 브라우징 (판타지, SF, 어반 판타지 등)
  • 좋아요와 댓글 기능으로 커뮤니티 피드백
  • 로어북 검색 및 필터링 시스템
  • 마크다운 형식 지원으로 풍부한 포맷팅
  • 이미지와 맵 첨부 기능
